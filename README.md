# medi_sleep_tracker
A meditation and sleep tracker application – project for Mobile Application Development (React Native + Expo).

## Requirement before you run

- Node.js (LTS)

## Run the projectproject

```bash
npm install
npm run start
```

After it you can do:

- Press `a` for open in Android (need Android Studio/emulator) or expo on your mobile phone to run it
- Press `w` for open web version
